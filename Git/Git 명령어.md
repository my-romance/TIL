# Git 명령어 정리

### git 계정 확인 및 변경

```shell
# git 계정 확인
git config user.name # git 현재 사용자 이름 확인
git config user.email # git 현재 이메일 확인
```

```shell
# git 계정 변경
git config --global user.name <user name>
git config --global user.email <user.email>
```
